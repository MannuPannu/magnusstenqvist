Tech stuff

I started on this site to learn more about angularjs and node. And thats the best advice I can give people, if you want to learn something; take a concept and do it in the technology you want to learn. But do it small. The first version of this site was basically one view and a controller who printed out some dummy texts. And I went from there. Now I have a simple site (yet it is working!) where I can put up blog posts wherever I am.

This site consist of:

  • Angular in the frontend
  • A node server as backend + express
  • A mongodb database
  • Passportjs as authentication manager for my admin part
  • Bootstrap for instant good looks (albeit common everywhere)
  • This site is hosted on Red Hats cloud service Open Shift. You get like 3 free sites on their cloud service, includingmongodb
